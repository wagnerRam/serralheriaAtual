# Serralheria Atual - Painel Admin

Este é um sistema de gerenciamento para a **Serralheria Atual**, desenvolvido em **Next.js + TypeScript**.  
O projeto tem como objetivo centralizar e facilitar a gestão de orçamentos e clientes através de um **painel administrativo moderno**.

---

## Tecnologias Utilizadas
- [Next.js](https://nextjs.org/) (React Framework)
- [TypeScript](https://www.typescriptlang.org/)
- [TailwindCSS](https://tailwindcss.com/) (estilização)
- [Shadcn/UI](https://ui.shadcn.com/) (componentes prontos)
- [Lucide Icons](https://lucide.dev/) (ícones)
- [PostCSS](https://postcss.org/)

---

## Pré-requisitos

Antes de rodar o projeto, você precisa ter instalado:
- [Node.js](https://nodejs.org/) (versão 18 ou superior recomendada)
- [npm](https://www.npmjs.com/) ou [yarn](https://yarnpkg.com/)

---

## Como Rodar o Projeto Localmente

1. Clone este repositório:
   ```bash
   git clone https://github.com/wagnerRam/serralheriaAtual.git

2. Acesse a pasta do projeto:
   ```bash
   cd serralheriaAtual

3. Instale as Dependências:
   ```bash
   npm install

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev

5. Abra no navegador http://localhost:3000
   ```bash
   http://localhost:3000
